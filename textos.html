Párrafos

Este es un párrafo y como párrafo debo estar entre las etiquetas "p". En un párrafo puedes encontrar texto en negrita, para eso tienes que hacer uso de la etiqueta "b" o "strong". También puedes encontrar texto en cursiva, para eso tienes que hacer uso de la etiqueta "i" o "em". También puedes encontrar texto subrayado, con la etiqueta "u" o "ins", aunque esta última se debe usar para elementos que se están insertando al documento. Así como puedes subrayar, puedes tachar, usa la etiqueta "s" o "del", pero la segunda para indicar un elemento eliminado del documento.
